{% extends "page_base.html" %} {% block title %}Datasette Tutorials{% endblock %} {% block content %} {% markdown %} # Datasette Tutorials These tutorials are intended to complement the official [Datasette documentation](https://docs.datasette.io/) by providing step-by-step guides to learning Datasette and the Datasette ecosystem tools. - **[Exploring a database with Datasette](/tutorials/explore)** shows how to use the Datasette web interface to explore a new database. - **[Learn SQL with Datasette](/tutorials/learn-sql)** introduces SQL, and shows how to use that query language to ask questions of your data. - **[Using Datasette in GitHub Codespaces](/tutorials/codespaces)** shows how Datasette can be installed and used in free GitHub Codespaces browser-based development environments. - **[Cleaning data with sqlite-utils and Datasette](/tutorials/clean-data)** shows how to use sqlite-utils to clean data, import it into SQLite and then explore it with Datasette. Includes a ten minute accompanying video. - **[Building a location to time zone API with SpatiaLite](/tutorials/spatialite)** shows how to use SpatiaLite and Datasette to create an API for looking up time zones based on latitude/longitude points. If you're looking for something longer: - **[Data analysis with SQLite and Python](/tutorials/data-analysis)** is a 2hr45m tutorial presented at PyCon 2023.
